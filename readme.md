# Installation guide

## Required Packages

| Package | use |
| ------- | --- |
| Hyprland-git | Hyprland desktop, all of my configs are written using the git-version |
| rofi | Application Search |
| Ags (Aylurs gtk shell) | All widgets, and the bottom bar |
| Hyprshot | Screenshots (Optional) |
| Kitty | You can change this to something else, just make sure you have a way to access the files |
| Dolphin | Again, same default, you can change this but make sure you can access it | 
| Astal-Hyprland | Used to get the focused monitor, will be used to show the workspace soon |
| Astal-Tray | Used to get the Tray information | 
| Astal-Battery | Used to display the battery on laptops. (Its still required on desktops, unsurprisingly.) | 

Overall, just move all the folders **excluding hypr_nso_dotfiles, and wallpapers** into .config.

I am going to make a script for this.

## Fonts
Move the hypr_nso_dotfiles folder into ~/.local/share/fonts

## wallpapers
Move the wallpapers folder into ~/Pictures/
